DANILO'S ENTERPRISE BACKUP

Permite programar el respaldo de los archivos que las organizaciones consideren
como importantes, para eso se seleccionan las carpetas que se habrán de respaldar
así como el destino, pudiendo ser otra carpeta dentro de la misma computadora o
una unidad conectada en red; además se debe establecer el periodo en el cual se
requieren los respaldos.

El programa cuenta con la posibilidad de bloquearse con contraseña para que solo
el administrador de los equipos pueda acceder a estas configuraciones.
